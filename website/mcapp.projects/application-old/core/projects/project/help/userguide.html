  • User Start

    User Start

    When you login for the first time into Materials Commons we expect you will have no projects / data entered. User Start is intended to help all the new users who are trying to understand what MaterialsCommons does.
    Overview

    Provenance is the heart of the Materials Commons. The flowchart below is designed to help user understand the workflow of how to create Provenance. Areas highlighted in grey are the essential steps to create provenance

    Create Project

    In-order to upload files first you have to create project(or workspace). To create a project click on drop-down list and hit "New Project" which will ask you to enter name for the project. You can create multiple projects which means evry project will have its own workspace

    Why you have to create project ?

    Project is like your workspace where you can store files , notes, samples , provenance related to that project.

    Upload files

    Under the files panel you will see an empty project folder is created. Now you can click the upload icon to upload your files into the project.

    Why you have to upload files ?

    Prior to creating the provenance (refer to the flowchart) you will have to upload your data onto Material Commons. So that this uploaded data will be available during provenance creation.

    Create Sample

    Under the samples panel click on icon and fill in the details of the sample and hit create which will add sample to the list.

    Why you have to create sample ?

    Sample is a piece of material on which you wanted to perform some experiments. Similar to file uploads you have to create your sample so that it will be available for you to choose during provenance creation

    Associate Provenance

    Under the processes panel click on the icon to create process. Then from the drop-down list select what type of experiment/analysis you wanted to run and then associate your data at the input/output steps.

    Why you have to create provenance?

    The purpose of creating provenance is to keep track of the history of changes done to the data items.

  • Glossary

    Glossary

    Dashboard:
    is the area where you can see all panels for a project.
    Draft:
    can be defined as an incomplete provenance step. When you save your provenance as draft you can go back and modify at anytime.
    Last login:
    will provide your last login date onto Materials Commons.
    Project:
    is defined as a workspace where you can store all the information related to a project or perform some actions against them. Every project will have its own defined workspace.
    Provenance:
    is defined as the history of data
    Process:
    is one of the key step in provenance creation. Process is defined as an experiment / computation / data analysis with a set of inputs and outputs.
    Panel:
    is a container to list various items (such as reviews/samples/notes/processes/files) inside a project.
    Review:
    is defined as an assessment or examination of something with the possibility or intention of instituting change if necessary.
    Sample:
    is defined as a material on which user can perform some experiment. During provenance creation sample will be an input into the process.
    Template:
    is a placeholder for detailed steps inside a process(experiment/computation/data analysis). There are set of pre-defined templates on Materials Commons.
    Sideboard:
    is an area where you can add/drag various items to compare side-by-side.

    Global Reviews
    Draft

    Global reviews section will list all the active reviews from all the projects in your list. Global reviews are categorized into 2 types. 1) Requested 2) To Conduct.
    Requested reviews are the reviews you have created and assigned to other users. Whereas To Conduct reviews are the reviews which are assigned to you by other users. Detailed information about the reviews are discussed in reviews section. When you click on a particular review it will take you to the respective project under which the review is created.

  • Reviews

    Reviews

    Panel will list all the reviews created inside a project. List may contain reviews which you have requested to others or vice-versa. When you click on the title, review toggles and details will be opened. You can comment on the review by clicking on the comment button. If you close the review it will be removed from the list (archived).

    Draft
    How to create a review

    When you click Plus icon it will open up a form where you can fill in the title, pick a user who has to review and add your comment.

    Attach samples/files/notes to review

    There is an option to attach samples/files/notes along with your comment. In order to attach click on the + sign of the respective item which will trigger and open up an option to add those items to your review. Example: If you want to attach a note, click on the + sign next to notes and then you will see options in the notes panel as shown in picture below. When you hit create it will create a review for a project and will be notified to the assignee under global-reviews( which is in navbar).

    Draft
    How to create a sample

    When you click Plus icon it will open up a form where you can fill in the name, alloy supplier and sample type. Icons annotated as 1, 2 and 3 (Image) are explained below:

    1. Composition

    When you click composition button, it will open up dropdown list of elements using which you can add composition to the sample

  • Processes

    Provenance

    Provenance panel will list all the processes created for a project.

    Draft

    When you click on the process name, process details will be opened. Graph will briefly describe about what are the inputs to the process and outputs from the process.

    How to create a provenance:

    Step1 : Choose Template

    In order to create provenance first you have to choose a template. Based on the template you choose, provenance steps may vary. For Example: You wanted to collect data through Atom Probe Experiment. So now you have to choose APT Data Collection Template from the drop-down list

    After you choose the template it will will show you what are the steps involved to create a provenance.

    Step2 : Process Details

    Here you have to fill in the process name( Experiment/Computation) which is a required field. And fill out other details as necessary.

    Step2: Inputs

    Input Steps vary based on the template chosen. Here for the "APT Data Collection" template there are 3 sub steps under inputs. 1) Choose a sample 2) Settings 3) Files

    1. Choose a Sample

    You can choose a sample from the available using dropdown menu or create a new sample

    Settings

    Fill out settings as per the template

    Add Files

    Files can be added using files panel(project tree). All the files inside this panel will give you an option called "Add to process". By clicking this button you can add files to the provenance

    Step3: Outputs

    Same as inputs

    Step 4: Submit

    In this step you can review all your content by clicking on checkbox. Then submit the provenance. After provenance is submitted you cannot make any modifications.So if you are not sure about any step you can save it as a draft which will save it in the list of drafts and make modifications at a later point.
